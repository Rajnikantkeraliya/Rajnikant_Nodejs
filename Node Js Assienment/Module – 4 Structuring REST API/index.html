2. What is difference between mongo DB and SQL

MongoDB (NoSQL) SQL Database
Uses flexible JSON-like documents Organizes data into tables with rows and columns
Query Language: MongoDB Query Language (MQL) Query Language: Structured Query Language (SQL)
Flexible schema: Documents can have different structures Rigid schema: Tables must adhere to predefined structure
Designed for horizontal scalability Traditionally scaled vertically, but also supports horizontal scaling
Supports multi-document transactions, but sacrifices strict ACID compliance ACID-compliant, ensuring data integrity and consistency
